### 举个直观的例子：
1. **默认情况（带换行）**：
   ```python
   print("Hello")
   print("World")
   ```
   输出：
   ```
   Hello
   World
   ```

2. **设置 `end=""`**：
   ```python
   print("Hello", end="")
   print("World", end="")
   ```
   输出：
   ```
   HelloWorld
   ```
